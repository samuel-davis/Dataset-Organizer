# Deep Learning Framework Data Organizer

# Installation
- Java JDK 7 or higher must be installed

# Building
```
mvn clean install
```
# Running the Program
- Simply
```
java -jar DataOrganizer-1.0.jar Images/
```
OR
```
java -jar DataOrganizer-1.0.jar /home/sam/Pictures/Images/
```
